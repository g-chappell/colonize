# Colonize — Phase 1 Epic Draft

Produced by `/pm-brainstorm` during `/init-autonomous` Phase 1 (2026-04-20).
Consumed by `/roadmap-expand` in Phase 6 to write `roadmap/roadmap.yml`.

## Approved shape

- **MVP option:** C (Full-MVP, public-launch candidate)
- **All original Colonization mechanics preserved** (reskinned to OTK lore)
- **Playable factions (4):** Order of the Kraken · Ironclad Syndicate · Phantom Corsairs · Bloodborne Legion
- **Tithing authority:** The Rayon Concord (system-imposed, non-playable)
- **Victory conditions:** Prosperity · Discovery · Sovereignty · Conquest
- **MVP excludes:** multiplayer, sky, deep, Azure Dominion gameplay, Abyssal Horror combat

## Epic ladder

### Tier 1 — Foundation

**EPIC-01 — Platform foundation**
- Story: web app shell with dev/prod build
- Story: mobile wrapper (PWA + shop-deliverable native shell)
- Story: shared state/store layer usable across web + mobile
- Story: CI + deploy pipeline wiring

**EPIC-02 — Retro art pipeline & UI shell**
- Story: 2D pixel-art tile atlas pipeline
- Story: HUD chrome (resource bar, minimap, end-turn, year/era display)
- Story: Screen navigation (main menu → faction select → in-game → pause)
- Story: Audio hooks (BGM, SFX buses; sea/tavern/combat layers)

### Tier 2 — Core gameplay

**EPIC-03 — World & map** *(depends on 01, 02)*
- Story: procedural map gen (all tiles fixed at gen; ocean, floating-city nodes, islands, Red Tide zones, Fata Morgana)
- Story: fog of war + line-of-sight
- Story: camera, zoom, pan, mobile touch controls
- Story: Rayon Passage start-region (always revealed at start)

**EPIC-04 — Turn engine, units & movement** *(depends on 03)*
- Story: turn loop (player → AI turns → world events → next year)
- Story: unit entities + stats (crew, supplies, condition)
- Story: ship classes (Sloop/Brig/Frigate/Ship-of-the-Line + OTK Legendary slots)
- Story: pathfinding + sailing rules (winds, currents, Red Tide impassability)

**EPIC-05 — Exploration & discoveries** *(depends on 04)*
- Story: rumour-tile mechanic (Archive caches, wrecks, Kraken shrines, Fata Morgana outcomes)
- Story: treasures (Salvage tech, Abyssal materia, bloodline artifacts)
- Story: legendary shipwreck discovery → OTK blueprint unlock / other factions get salvage
- Story: scouting unit types (Cartographer, Explorer)

**EPIC-06 — Colonies, resources & professions** *(depends on 04)*
- Story: colony founding (floating-city vs island-colony differences)
- Story: building tree (Tavern, Shipyard, Forge, Archive-study Hall, Chapel-of-Kraken, Distillery, Gun Deck, etc.)
- Story: tile-work assignment + resource production
- Story: resource chains (Timber→Planks; Rum; Forgework; Salvage; Abyssal materia)
- Story: profession system (crew roles, training via schools/Archive/shrines/capture)

**EPIC-07 — Trade** *(depends on 06)*
- Story: inter-colony cargo hauling (manual + automated routes)
- Story: home-port trade screen (Port Royal for OTK; equivalents per faction)
- Story: price dynamics (supply/demand on home-port goods)
- Story: merchant vessel automation (ghost routes)

**EPIC-08 — Combat & warfare** *(depends on 04, 06)*
- Story: ship-vs-ship resolution (hull, crew, guns, morale; ramming, broadsides, boarding)
- Story: ground combat on islands (Marine, Dragoon-analog, Cannon)
- Story: raiding (privateer attacks on trade routes)
- Story: fortifications (colony defences, ramparts, harbour booms)

**EPIC-09 — Diplomacy & faction stances** *(depends on 08)*
- Story: rival-faction diplomacy (war, peace, alliance, trade pact, tribute)
- Story: Abyssal-site stance system (worship / salvage / guard / exploit)
- Story: NPC encounter factions (Pale Watch patrols, Abyssal Brotherhood cult raids, Sons of Scylla guerrillas, Scarlet Forge tech-traders, Blackwater Collective smugglers)
- Story: faction-unique bonuses for the four playables

### Tier 3 — Meta, narrative, release

**EPIC-10 — Concord meta-layer & Sovereignty endgame** *(depends on 06, 09)*
- Story: Liberty Chimes → Elders Council → Archive Charters (Founding-Fathers analog; 2-choice selection on each event)
- Story: Rayon Concord tithes (rising passive drain; tension meter)
- Story: Tidewater Party boycott event (dumps cargo, zeroes tension briefly, escalates Concord response)
- Story: Declaration of Sovereignty → Concord Fleet spawn + invasion
- Story: victory-condition engine (Prosperity · Discovery · Sovereignty · Conquest) + loss states

**EPIC-11 — Narrative, tutorial & flavour integration** *(depends on most of Tier 2)*
- Story: Sparrow's Diary prologue (static/cinematic opener; player assumes successor role)
- Story: tavern rumour system (procedural + hand-authored; Port Royal/Tortuga/Barataria flavour)
- Story: Codex / in-game lore browser (OTK Universe Reference entries; `[OPEN]` items surfaced as fragmentary)
- Story: tutorial campaign (first 20 turns coach founding/trade/combat/Archive/Sovereignty prep)
- Story: three-register flavour-text pipeline (salt-and-rum · eldritch · salvaged-futurism; tagged per UI surface)

**EPIC-12 — Monetization, accounts, cloud save, store release**
- Story: optional account system + cloud save (syncs across web/mobile on same account)
- Story: ad integration (interstitial every N turns, configurable; never during Sovereignty War or narrative beats)
- Story: IAP — Remove Ads + cosmetics-only
- Story: store listings (web marketing site, App Store, Play Store), privacy policy, age rating, screenshot reel, launch trailer
- Story: submission + review response

## Post-MVP expansion flags (NOT in roadmap now)

- Multiplayer (hot-seat, async, real-time)
- Sky gameplay (Azure Dominion on-stage)
- Deep / underwater gameplay
- Abyssal Horror awakening + boss combat
- Additional playable factions
- Campaign arcs beyond NW 2191 opener
